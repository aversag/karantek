# Rôle : Expert QA & Testing

## 1. Mission
Garantir la qualité du site avant chaque déploiement via une checklist de tests systématiques.

## 2. Tests Fonctionnels

### A. Navigation
- [ ] Tous les liens internes fonctionnent.
- [ ] Pas de liens cassés (404).
- [ ] Menu responsive fonctionne sur mobile.
- [ ] Ancres (#) scrollent correctement.

### B. Formulaires
- [ ] Validation fonctionne (champs requis).
- [ ] Messages d'erreur clairs et visibles.
- [ ] Soumission envoie bien l'email.
- [ ] Message de confirmation affiché.
- [ ] Honeypot anti-spam actif.

### C. Contenu
- [ ] Pas de Lorem Ipsum restant.
- [ ] Images toutes chargées.
- [ ] Textes sans fautes d'orthographe.
- [ ] Informations de contact à jour.

## 3. Tests Cross-Browser

### A. Navigateurs
- [ ] Chrome (dernière version).
- [ ] Firefox (dernière version).
- [ ] Safari (Mac/iOS).
- [ ] Edge.

### B. Devices
- [ ] Mobile portrait (375px).
- [ ] Mobile paysage.
- [ ] Tablette (768px).
- [ ] Desktop (1280px+).

## 4. Tests Performance

### A. Lighthouse
- [ ] Performance > 90.
- [ ] Accessibility > 90.
- [ ] Best Practices > 90.
- [ ] SEO > 90.

### B. Core Web Vitals
- [ ] LCP < 2.5s.
- [ ] FID < 100ms.
- [ ] CLS < 0.1.

## 5. Tests Accessibilité

- [ ] Navigation au clavier complète.
- [ ] Focus visible sur tous les éléments.
- [ ] Contrastes suffisants.
- [ ] Alt text sur images.
- [ ] Lecteur d'écran : structure logique.

## 6. Tests Sécurité

- [ ] HTTPS actif.
- [ ] Headers de sécurité présents.
- [ ] Pas de données sensibles exposées.
- [ ] Console sans erreurs.

## 7. Validation Pré-Déploiement
> *"Ai-je testé sur mon téléphone personnel ? La page d'accueil charge-t-elle en moins de 3 secondes ?"*
