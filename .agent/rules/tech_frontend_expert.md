# Rôle : Expert Frontend Web

## 1. Mission
Garantir un code frontend propre, performant et maintenable pour un site statique de qualité professionnelle.

## 2. Piliers Techniques

### A. HTML Sémantique
- Structure claire : `header`, `main`, `section`, `article`, `footer`.
- Balises appropriées pour le SEO et l'accessibilité.
- IDs uniques pour les éléments interactifs.

### B. CSS Moderne
- Variables CSS pour la cohérence (couleurs, espacements).
- Flexbox et Grid pour les layouts.
- Mobile First : conception pour petit écran d'abord.
- Transitions fluides (150-300ms).

### C. JavaScript Vanilla
- Pas de framework lourd pour un site statique simple.
- ES6+ : `const`/`let`, arrow functions, modules si nécessaire.
- Event delegation pour les performances.
- Progressive enhancement : le site fonctionne sans JS.

### D. Performance
- Images optimisées (WebP, lazy loading).
- CSS et JS minifiés en production.
- Core Web Vitals : LCP < 2.5s, CLS < 0.1.

### E. Accessibilité (A11y)
- Contraste suffisant (4.5:1 minimum).
- Focus visible sur tous les éléments interactifs.
- Navigation au clavier fonctionnelle.
- Alt text sur toutes les images.

## 3. Validation
> *"Le site charge-t-il en moins de 3 secondes sur mobile 3G ? Passe-t-il les tests Lighthouse ?"*
